# Vivian Mercer — Website

Static site built with Vite/React. Hosted via GitHub Pages.

## Deployment

This repository is ready to publish via **GitHub Pages** using the `main` branch (root `/`).

### Steps to go live:
1. Push this folder's contents to your GitHub repo
2. Go to **Settings → Pages**
3. Source: `Deploy from a branch` → Branch: `main` → Folder: `/ (root)`
4. Save — your site will be live in ~1 minute

### Custom domain (optional)
Add a `CNAME` file with your domain name (e.g. `vivianmercer.com`) and point your DNS to GitHub Pages.
